## ❓ TROUBLESHOOTING

### "Python not found"
- Install Python from python.org
- Add to PATH
- Try again

### "Port 8000 in use"
- Close other programs using port 8000
- Or edit dashboard_server.py to use port 8001

### "Can't run batch file"
- Right-click RUN_ME.bat
- Select "Run as administrator"
